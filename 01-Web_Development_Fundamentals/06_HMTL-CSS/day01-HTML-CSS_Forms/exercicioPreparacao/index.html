HTML & CSS - Forms

Observações durante o exercício

  • Autofocus: assim que a tela carrega, deixa ativo o campo com esta propriedade.
  • O texto que vai dentro do botão precisa ser incluído dentro da propriedade value.
  • O texto de um checkbox precisa vir dentro de um label específico (com a id na propriedade 'for').
  • Tornar label clicável: incluir o id do elemento na propriedade for de sua label - ver checkbox.
